## In Short
</div>

**Like Zustand, but has additional capabilities:**

- No selectors: automatically optimizes re-renders
- Store events: `onFirstSubscribe`, `onSubscribe`, `onUnsubscribe`, `onLastUnsubscribe`
- Easier to set the initial state on SSR/SSG
- Smaller bundle

**Async support like TanStack Query, but:**

- DX is very similar to Zustand: Store is available globally, access/modify state from anywhere
- Better docs: [concept](/docs/query-concepts) & [state transition](/docs/query-state-machine) are clearly explained
  <div className='opacity-70 text-sm'>(honestly, many developers have used TanStack Query for years but still don't fully understand the difference between `refetch` and `invalidate`)</div>
- Much smaller bundle than TanStack Query
